📦 Banner to DocStar Extension

Western Oregon University

Choose Your Operating System

⚠️ IMPORTANT: Your browser may block some downloads (.js, .bat, .sh files) as potentially unsafe. You must click "Keep" for each blocked file to complete the installation. This is normal - these files are safe and required for the extension to work.
🪟
Windows
For Windows 10/11 with Chrome or Edge
🍎
macOS
For Mac with Chrome or Edge
ℹ️ What happens when you click download:
  • Extension files will be downloaded to your computer
  • Files will be placed in the BannerExtension folder
  • Step-by-step installation instructions will appear below
📥 Files Downloaded! Check your Downloads folder.
⚠️ Did your browser block some downloads?
Windows browsers often block .js, .bat files as "potentially unsafe."
Click "Keep" on each blocked file - they are safe and needed for the extension.
1 Run the Move Script
Go to your Downloads folder and double-click:
move_files.bat
This will automatically move all extension files to C:\BannerExtension
⚠️ Important: You must run the move_files.bat script before proceeding. The extension won't work until files are moved to C:\BannerExtension.

After running move_files.bat, choose your browser:

🌐
Chrome
🔷
Edge
2 Open Chrome Extensions Page
Copy this URL and paste it into Chrome's address bar:
chrome://extensions/
3 Enable Developer Mode
Toggle ON "Developer mode" in the top-right corner.
4 Load the Extension
Click "Load unpacked" and paste this path:
C:\BannerExtension
5 Enable File Access
Click "Details" on the extension, then toggle ON "Allow access to file URLs".
6 Pin to Toolbar
Click the puzzle piece icon 🧩 in Chrome's toolbar (top-right), then click the pin icon 📌 next to "Banner to DocStar" to keep it visible in your toolbar.
🎉

Installation Complete!

The extension is ready to use in Chrome.

📌 First Time Using the Extension?
The first time you click the extension, you'll need to:
1. Login to DocStar when prompted
2. Quickly close the welcome popup within 2 seconds (appears after login)
3. The invoice will appear automatically

After the first time, the extension works instantly with no popup!

📥 Files Downloaded! Check your Downloads folder.
⚠️ Did your browser block some downloads?
Safari/Chrome may block .js, .sh files as "potentially unsafe."
Click "Keep" or "Allow" on each blocked file - they are safe and needed for the extension.
1 Run the Move Script
Option A - Terminal (Recommended):
  1. Open Terminal (in Applications → Utilities)
  2. Type: bash ~/Downloads/move_files.sh
  3. Press Enter
Option B - Finder (if Terminal doesn't work):
  1. Go to your Downloads folder
  2. Right-click move_files.sh
  3. Choose "Open With" → "Terminal"
⚠️ Important: The script will automatically move all extension files to ~/BannerExtension. Don't proceed to the next step until this completes.

After running move_files.sh, choose your browser:

🌐
Chrome
🔷
Edge
2 Open Chrome Extensions Page
Copy this URL and paste it into Chrome's address bar:
chrome://extensions/
3 Enable Developer Mode
Toggle ON "Developer mode" in the top-right corner.
4 Load the Extension
Click "Load unpacked" and navigate to:
~/BannerExtension
💡 Mac Tip: Press Cmd+Shift+G in the folder browser, paste the path above, and press Enter.
5 Enable File Access
Click "Details" on the extension, then toggle ON "Allow access to file URLs".
6 Pin to Toolbar
Click the puzzle piece icon 🧩 in Chrome's toolbar (top-right), then click the pin icon 📌 next to "Banner to DocStar" to keep it visible in your toolbar.
